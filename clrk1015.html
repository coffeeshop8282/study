커뮤니케이션 기초
Communication 정의 (암기)
정보(information)를 발신자(source)와 수신자(receiver) 사이에서 전달하는 과정
방법: 언어(verbal) + 비언어(non-verbal)
성공 조건: 수신자가 발신자의 의도대로 해석해야 함
2026 디지털 환경
장점 Benefits
정보(Information) 증가
연결성(Connectivity) 증가
단점 Disadvantages
콘텐츠 경쟁 증가
건강 문맹(Health Illiteracy)
가짜 뉴스 · 반지성주의(Anti-intellectualism)
NOISE — 커뮤니케이션 방해 요소
커뮤니케이션 완전 모델 + Noise 심리 3가지
Sender · Encoding · Message · Medium · Receiver · Decoding · Context · Noise
Noise 심리적 요인 3가지 (암기):
• Confirmation Bias (확증 편향) — 기존 믿음을 확인하는 정보만 선택적 수용
• Sunk Cost Fallacy (매몰 비용의 오류) — 이미 투자한 것 때문에 잘못된 결정 지속
• Availability Heuristic (가용성 발견법) — 쉽게 떠오르는 정보를 더 중요하게 여기는 편향
커뮤니케이션 모델 2가지
Linear Model — 선형 모델
일방향 · Sender → Receiver · 발신자 중심
구성: Sender → Message/Channel → Receiver
특징: 발신자(Sender) 중심, 수신자는 "목표점"만, 상호작용 없음
한계: 실제 소통은 이보다 복잡 (Noise 존재, 피드백 없음)
Transaction Model — 거래 모델
양방향 · 동시 쌍방 소통 · 의료 현장에서 사용
핵심 차이: Sender/Receiver → Communicators (동시에 역할 수행)
메시지를 보내면서 동시에 상대방 반응 수신 → 실시간 조정
목적: 관계 형성 / 동맹 구축 / 자아 개념 형성 / 참여
핵심 차이 (암기)
Linear = 일방향, Sender 중심, 수신자는 끝점
Transaction = 양방향, 동시, 참여자 = Communicators, 실시간 조정 가능
Academic Writing vs Technical Writing
항목Academic WritingTechnical Writing
목적지식 시연교육·설득 · action/results driven
지식 수준독자(강사)가 더 많이 앎작성자가 더 많이 앎
독자강사 1명, 처음부터 끝까지 읽음다수, 다양한 배경과 지식 수준
평가 기준깊이·논리·맞춤법·문법독자의 필요 충족 (명확·간결)

Responsibility vs Accountability (암기!)
Responsibility vs Accountability 차이
시험 자주 출제
Responsibility = 수행하는 업무·과제 (tasks you perform)
Accountability = 결과에 대해 사람에게 답해야 하는 의무

"You are responsible for things, but accountable to people for the outcomes."

설득 (Persuasion) & 수사학 (Rhetoric)
핵심 전제 (암기)
Technical Writing = action/results driven → 행동을 이끌어내야 함
이를 위해 설득(Persuasion)이 필수 → 도구 = Rhetoric(수사학)
설득의 4가지 공통 특징
Persuasion의 정의와 특징
설득자가 피설득자의 정신 상태를 변화시키는 것
1. 설득자가 피설득자에게 영향을 주려 의도함
2. 효과는 커뮤니케이션(언어=Rhetoric)으로 달성
3. 성공의 개념 내포됨
4. 피설득자에게 자유의지(free will) 존재

행동 변화를 위해 먼저 생각(think)을 바꿔야 함
Rhetoric 정의 (아리스토텔레스)
언어를 사용해 청중의 생각·감정·행동을 변화시키는 기술
"실용적 토론 도구 — 일반 청중을 설득해 실용적 문제를 해결하는 수단"

진정한 수사학을 위해:
• 지식(Knowledgeable) · 윤리(Ethical) · 높은 도덕적 성품

수사학의 과정: ① 의식적 ② 전략적 ③ 특정 결과 의도 ④ 기술(Skill) 필요
항상 타겟 청중의 입장에서 스스로를 바라볼 것
